# vue-isud
ISUD with Vue.js and JSON as databases stored on the local system.

## Project setup
```
npm install
```

### Run JSON Server
```
json-server db.json
```

### Compiles and hot-reloads for development
```
npm run serve
```
open http://127.0.0.1:8080

### Compiles and minifies for production
```
npm run build
```
